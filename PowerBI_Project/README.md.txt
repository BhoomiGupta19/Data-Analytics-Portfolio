# Sales Analytics Dashboard – Power BI

## Project Overview
This project focuses on building an interactive Sales Analytics Dashboard using Microsoft Power BI to analyze sales performance, profitability, and discount impact across multiple business dimensions.

The dashboard is designed for business decision-making, enabling users to explore key metrics dynamically through filters and visuals.


## Tools & Technologies
- Microsoft Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)


## Dataset
- Real-world sales dataset containing information on:
  - Sales
  - Profit
  - Discount
  - Product Category & Sub-Category
  - Customer Segment
  - Region
  - Shipping Mode


## Data Preparation
- Cleaned and transformed data using **Power Query**
- Created derived columns:
  - Profit Margin
  - Discount Flag
- Ensured correct data types for financial analysis


## DAX Measures Created
- Total Sales
- Total Profit
- Average Profit Margin

These measures dynamically update based on filters and slicers.



## Dashboard Features
- KPI Cards
  - Total Sales
  - Total Profit
  - Average Profit Margin
- Visual Analysis
  - Sales by Category
  - Profit by Region
  - Discount Impact Analysis
- Interactivity
  - Slicers for Region, Segment, Category, and Ship Mode
  - Cross-filtering across all visuals


## Key Insights
- Identified top-performing product categories by revenue
- Highlighted the most profitable regions
- Analyzed how discounts affect sales across customer segments
- Provided an executive-level overview through KPI metrics


## Project Outcome
This dashboard demonstrates:
- Business-oriented data analysis
- Strong understanding of Power BI fundamentals
- Ability to translate raw data into actionable insights

The project is suitable for **Business Analytics roles** and **MSc Business Analytics applications**.
