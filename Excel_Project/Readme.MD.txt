Advanced Excel Project: Superstore Sales & Profitability Analysis

Objective
To analyze retail sales data and identify key drivers of profitability using advanced Excel techniques.

Tools Used

Microsoft Excel

Pivot Tables

Conditional Formatting

Advanced Formulas

Key Analysis Performed

Category and region profitability analysis

Impact of discounting on margins

Customer segment performance evaluation

Operational efficiency through shipping mode analysis

Key Insights

Profitability varies significantly across categories and regions.

Excessive discounting negatively impacts margins.

High-revenue customer segments may not always be the most profitable.

Logistics choices directly affect profitability.

Skills Demonstrated

Data cleaning and transformation

Financial metrics (profit margin)

Advanced pivot table analysis

Business insight generation

Dashboard design
